Features

Easily Customizable

Himeji Map Viewer comes with a variety of options to quickly - and easily - customize your render. HMV Has four main options:

  • Biome Colors
  • Night Brightness
  • Shadow Intensity
  • Water Transparency

Each option can be customized in multiple ways, or be turne off entirely. Please see the next section for details for each one.

Along with tweaking how the image will render, you can choose to render only a certain area of your map.

Light Mod Support

If a modded map is selected, Himeji Map Viewer will do its best to work around unknown blocks and biomes. You can select torender unknown blocks in Magenta, or to skip them and render the first known block below it. Biomes will be rendered using a default biome selected in the settings.

After rendering, HMV will alert you if any unknown blocks or biomes were found. You may choose to import these missing blocks and biomes so that you can assign colors to them. Biome names are not stored in the file, so HMV will tell you where to go in your world to find the biome name.

*Only modded maps from version 1.13 onwards will work properly.

Version Compatibility

Himeji Map Viewer supports all versions between Beta 1.3 and 1.16.1. HMV will be able to handle minor future updates that do not change how the world files are saved by allowing you to manually enter new blocks and biomes, reducing the need for constant updating as snapshots begin rolling out.

If there is an issue rendering a new version of Minecraft, please send a report so that the issue can be resolved as soon as possible.

Render Options

Biome Colors

Change what colors to tint grass, foliage, and water, along with how strong the tint should be.

Night Brightness

Adjust how dark nights will be. Light will scale between full brightness and the set darkness.

Shadow Intensity

Set how bright inclines will be and how dark declines will be. Settings can be locked to change together

Water Transparency

Adjust how transparent water will be. Water can be made completely opaque to hide what is underneath it.
